# 表情识别系统 - 功能说明文档

> 🌐 语言切换：[English Version](README_EN.md)

## 一、系统概述
本程序是一款基于 `tkinter` 搭建可视化界面、结合 OpenCV 人脸检测与**开发者基于公开面部表情数据集在个人电脑上自主训练**的 TensorFlow/Keras 深度学习模型的实时表情识别系统。程序可调用电脑摄像头，实时捕捉画面中的人脸并识别其表情类别，同时展示各类表情的概率占比，最终通过图形化界面直观呈现识别结果。

## 二、核心功能
| 功能模块 | 详细说明 |
|----------|----------|
| 可视化GUI界面 | 基于tkinter构建美观的图形化界面，包含标题、摄像头画面显示区、识别结果展示区、退出按钮 |
| 实时摄像头采集 | 调用电脑本地摄像头（默认索引0），实时采集视频画面并进行预处理（镜像翻转+固定尺寸缩放） |
| 人脸检测 | 加载OpenCV官方人脸检测分类器（haarcascade_frontalface_default.xml），精准检测画面中的人脸并绘制蓝色矩形框标注 |
| 多类别表情识别 | 基于**从公开数据集自主训练的Keras模型（best_emotion_model.h5）**，识别7种核心表情：愤怒、厌恶、恐惧、开心、中性、悲伤、惊讶 |
| 概率量化展示 | 计算并展示每种表情的概率占比（保留2位小数），同时标注概率最高的“当前表情” |
| 优雅退出机制 | 点击“退出”按钮可释放摄像头资源并关闭GUI窗口，避免资源占用 |
| 跨环境兼容 | 兼容Python开发环境与打包后的可执行文件（dist目录），自动识别运行环境并适配文件路径 |

## 三、技术栈
| 类别 | 技术栈 |
|------|--------|
| 界面框架 | tkinter（Python内置GUI库） |
| 计算机视觉 | OpenCV（摄像头操作、人脸检测、画面预处理） |
| 图像处理 | PIL（将OpenCV图像转换为tkinter可显示格式） |
| 深度学习 | TensorFlow/Keras（加载自主训练模型、表情预测） |
| 数值计算 | NumPy（数据格式转换、张量处理） |
| 系统适配 | sys/os（识别运行环境、文件路径处理） |

## 四、模型训练背景
### 4.1 训练环境
- 硬件：个人电脑（基于实际训练配置的CPU/GPU）
- 软件：Python 3.7~3.10、TensorFlow 2.x、NumPy、Pandas、OpenCV
- 训练数据集：公开面部表情数据集（如FER-2013、CK+、JAFFE等）

### 4.2 训练流程
1. 数据集预处理：对公开数据集进行清洗，将人脸图像归一化为48×48灰度图，划分训练/验证/测试集
2. 模型构建：搭建适用于表情识别的卷积神经网络（CNN），包含卷积层、池化层、全连接层
3. 模型训练：在个人电脑上完成模型训练，调整超参数（学习率、批次大小、训练轮数）优化识别准确率
4. 模型优化：采用dropout、批量归一化等技术防止过拟合，将表现最优的模型保存为`best_emotion_model.h5`
